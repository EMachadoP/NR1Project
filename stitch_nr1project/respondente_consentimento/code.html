Uso de Dados e Ética

Informamos que os dados coletados através deste questionário são processados de forma totalmente anônima. A Institutional Intelligence utiliza protocolos de criptografia de ponta a ponta para garantir que nenhuma resposta possa ser vinculada à sua identidade pessoal.

As informações fornecidas serão agregadas a um banco de dados macro-estrutural para a geração de relatórios de risco e indicadores de desempenho institucional, em estrita conformidade com a Lei Geral de Proteção de Dados (LGPD).

"Nossa arquitetura de dados prioriza a segurança estrutural acima de tudo, eliminando riscos de vazamento ou exposição individual."
